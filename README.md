# systemchecker

This PHP script can tell you wich browser and operating system you are using.

Currently it can only detect the following browsers:
- Chrome
- Firefox
- Safari
- Opera
- Iceweasel
- Opera Mini
- Blackberry browser
- Android Webkit Browser (Stock Android Browser)
- Internet Explorer
- Edge

And this operating systems:
- Windows (2000+)
- Mac OS X
- Android
- Windows Phone
- IOS
- Blackberry OS
- Chrome OS
- Ubuntu
- Linux Mint
- Linux
- Unix
